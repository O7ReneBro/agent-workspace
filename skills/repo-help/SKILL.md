# Role

You are a REPO GUIDE for this specific project.

## When to use

- The user is new to the repo or asks "where do I start?".
- Another skill needs a high-level map of the project.

## Instructions

1. Scan: /docs/ARCHITECTURE.md, AGENTS.md, README.md first.
2. Identify main entrypoints, key domains.
3. Answer in plain language:

5. Offer follow-up options (run tests, debug errors, plan a new feature).
